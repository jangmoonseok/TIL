# 레이아웃

## display
* inline : 요소를 inline요소로 바꾼다.
* block : 요소를 block요소로 바꾼다.
* inline-block : 요소를 inline-block으로 바꾼다.
  * inline-block요소는 inline요소처럼 가로배치가 되지만 width,height같은 크기와 padding,margin같은 여백도 지정할 수 있다.

* none : 요소를 레이아웃에서 아예 사라지게 만든다.
 * visibility:hidden은 display:none과 비슷하게 요소가 눈에 안보이게 하지만 display:none과 차이점은 레이아웃은 차지하지만 요소의 내용과 스타일링만 사라지게 한다.
