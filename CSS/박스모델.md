# 박스모델

* content : 콘텐츠가 표시되는 영역 width,height로 크기를 적용할 수 있다.
* padding : 콘텐츠와 테두리(border)사이의 여백
* border : padding과 margin사이의 테두리
* margin : 가장 바깥 쪽 레이어로 박스와 다른 요소 사이 공백 역할을 한다.

## 크기

* block요소는 너비,높이 값에 따라 높이 너비가 달라지지만 inline요소는 너비,높이 속성을 적용할 수 없다.
* width : 콘텐츠의 너비
* height : 콘텐츠의 너비
* min-width,height : viewport나 부모요소의 영향을 받아 설정한 값보다 크기가 작아져도 값 이하로 크기가 줄어들지 않는다.
* max-width,height : viewport나 부모요소의 영향을 받아 설정한 값보다 크기가 커져도 값 이상으로 크기가 커지지 않는다.

## 여백

### margin 
* 네 방향 바깥 여백 영역을 설정하는 단축 속성으로 margin-top,right,bottom,left 속성을 단축시킨것이다.

```
1. margin: 10px; 
  /* 네 방향으로 10px여백이 생긴다 */

2. margin: 20px 40px;
  /* 위/아래에 20px여백, 오른쪽/왼쪽에 40px여백이 생긴다 */

3.  margin: 20px 40px 30px;
  /* 위에 20px여백, 오른쪽/왼쪽에 40px여백, 아래에 30px여백이 생긴다.  */

4.   margin: 20px 40px 30px 50px;
  /* 위20px, 오른쪽40px, 아래30px, 왼쪽50px 여백이 생긴다. */
```
* 값의 단위를 %로 설정 할땐 여백의 크기가 부모의 width에 따라 상대적으로 바뀌는걸 고려해야한다.

### 마진 상쇄(margin collapsing)
* 마진 상쇄, 마진 겹침, 마진 중복 등등 으로 불린다.
* 여러 블록요소들의 위/아래 여백이 가장 큰 크기를 가진 여백으로 결합되는 현상

1. 인접 형제 :  형제 요소의 위/아래 여백이 만나 상쇄된다.

2. 부모-자식요소 : 부모 요소에 border,padding,inline-content 모두 없을때 부모 요소의 여백과 자식 요소의 여백이 만나 더 큰 값으로 상쇄된다.

3. 빈 블록 : 블록의 border,padding,content가 없고 heigth또한 0 일때 해당 블록의 margin-top과 margin-bottom이 만나 상쇄된다.

### padding
* content와 border사이 여백을 나타낸다.(margin은 바깥여백 padding은 안쪽여백으로 생각하면 쉽다)
* margin과 같은 단축속성으로 사용 방법도 같다.
* 값으로 음수 값은 사용할 수 없다.
* 값의 단위를 %로 설정 할땐 여백의 크기가 부모의 width에 따라 상대적으로 바뀌는걸 고려해야한다.

## 테두리(border)
요소의 테두리를 설정하는 단축속성으로 border-width, border-style, border-color값을 설정한다.

### border-style 
* 테두리의 스타일을 정의한다.
* margin,padding과 같은 단축속성으로 사용방법도 같다.

### border-width
* 테두리의 두께를 정의한다.
* margin,padding과 같은 단축속성으로 사용방법도 같다.

### border-color
* 테두리의 색상을 정의한다.
* margin,padding과 같은 단축속성으로 사용방법도 같다.

### border-radius
* 요소 테두리의 꼭짓점을 둥글게 만든다.
* border속성이 없어도 정의할 수 있다.
* margin,padding과 같은 단축속성으로 사용방법도 같다.
* 값을 %로 정의할때는 가로축 값은 요소 박스의 너비에 대한 백분율이고, 세로축 값은 박스의 높이에 대한 백분율이다

## box-sizing
* 요소의 너비와 높이를 계산하는 방법을 지정한다.

### content-box
* 요소의  width,height속성이 콘텐츠 영역만 포함하고 여백,테두리는 포함하지 않는다.
* 요소의 크기 = 콘텐츠의 크기(여백과 테두리는 크기 값에 포함시키지 않는다)
```
.parent {
  width: 400px;
  padding: 20px;
  border: 30px solid blue;
}
```
* 이때 요소의 너비는 500px

### border-box
* 요소의 width,height속성이 바깥여백을 제외한 여백과 테두리크기를 포함한다.
* 요소의 크기 = 테두리 + 안쪽 여백+ 콘텐츠 크기로 계산한다.
```
.parent {
  width: 400px;
  padding: 20px;
  border: 30px solid blue;
  box-sizing: border-box;
}
```
* 이때 요소의 너비는 400px
