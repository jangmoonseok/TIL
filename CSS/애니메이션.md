
# animation
* transition처럼 두가지 상태사이를 변환하는거와 달리 다수의 스타일을 전환하는 애니메이션을 적용한다.
* keyframe으로 애니메이션 이름을 지정한다.

## @keyframes
* 애니메이션 중간중간의 특정 지점들을 제어할 수 있는 키프레임을 설정한다.
* 두가지 스타일만 적용할 경우 from to로 작성하고, 여러 스타일일 적용할 경우엔 %로 작성한다.
```
.box {
  width: 100px;
  height: 100px;
  background-color: blue;
  animation: first-animation 2s linear alternate infinite;
}

@keyframes first-animation {
  50% {
    background-color: aqua;
    width: 300px;
  }
  100% {
    background-color: blue;
  }
}
```

## animation-name
* 대소문자를 구분하고 알파벳a~z ,숫자0~9, 특수문자는 _ , -만 사용해 이름을 작성한다.

## animation-duration
* 애니메이션이 한 사이클을 완료하는데 걸리는 시간을 지정한다.
* 음수값은 유효하지 않으며 선언하면 무시된다.

## animation-delay
* 애니메이션의 시작을 지연시킬 시간을 지정한다.
* 음수값을 가질수 있으며 음수값을 지정할 경우 시작은 바로 시작하지만 지정한 시간전에 이미 애니메이션이 시작한거 처럼 작동된다.

## animation-iteration-count
* 애니메이션의 반복 횟수를 지정한다.
* number를 값으로 지정하거나 키워드를 사용해 지정한다.

## animation-direction
* 애니메이션의 방향을 지정한다.
* normal : 기본값으로 정방향으로 재생
* reverse : 역방향으로 재생
* alternate : 시작은 정방향 돌아올땐 역방향으로 재생한다.
* alternate-reverse : alternate의 반대방향으로 재생한다.

## animation-play-state
* 애니메이션의 재생 상태를 지정한다.
* running : 기본값으로 애니메이션의 상태가 움직이는 상태
* paused : 애니메이션의 상태가 멈춰있는 상태

## animation-fill-mode
* 애니메이션 실행 전과 후에 대상에 스타일 적용 방법을 지정한다.
* forwards : 실행 된 애니메이션의 마지막 keyframe 상태를 유지한다.
* backwards : 실행 될 애니메이션의 첫번쨰 keyframe상태를 delay시간동안 유지한다.
* both : forwards, backwards모두 적용

```
div {
  width: 100px;
  height: 100px;
  border: 1px solid black;
  animation: first-animation 2s alternate 1s;
}

.box1 {
  animation-fill-mode: forwards;
}
.box2 {
  animation-fill-mode: backwards;
}
.box3 {
  animation-fill-mode: both;
}

@keyframes first-animation {
  0% {
    background-color: red;
  }
  50% {
    width: 300px;
  }
  100% {
    background-color: blue;
  }
}
```
![GIF 2021-09-12 오후 2-57-49](https://user-images.githubusercontent.com/64582209/132973935-7cfe91ed-206e-4755-8e92-4f1aed532651.gif)
* forwards로 설정한 박스는 box의 기본 스타일이 적용된 상태에서 시작해서 끝날때는 keyframe 100%상태를 유지한다.
* backwards로 설정한 박스는 1s의 딜레이 시간동안 keyframe의 0%상태를 유지하고 끝날때는 box의 기본 스타일이 적용된다.
* both로 설정한 박스는 1s의 딜레이 시간동안 keyframe의 0%상태를 유지하고 끝날때는 keyframe 100%상태를 유지한다.
