# 색상과 배경

## 색상

### 색상 표현 방법
* 키워드를 사용 ( blue, transparent 등)
* #+16진수 표기법(HEX) : 0~10, A~F를 6자리로 색상을 표현(ex:#000000, #FFFFFF)
* rgb(), rgba() : 0~255까지 숫자를 함수의 인자로넣어 색상을 표현,rgba는 한가지 인자를 더 받으며 투명도를 표현

### opacity
* 0~1사이의 숫자를 사용해 불투명도를 표현
* 색상을 표현하는 rgba()의 alpha값과 같이 불투명도를 표현하지만 rgba는 배경의 색에만 영향을 주고 opacity는 요소자체에 영향을 준다.

## 배경

### background-color
* 모든 색상 표현 방법을 사용해 값을 지정한다
* background-image를 사용했을때 이미지가 배경색을 덮는다.

### background-image
* 요소의 배경 이미지를 지정한다.
* url("이미지 경로")로 작성한다.
* 이미지의 크기와 요소의 크기가 맞지 않으면 이미지가 잘리거나 반복될 수 있다.

### background-repeat
* 배경 이미지의 반복 방법을 지정한다.
* 기본값은 repeat으로 요소의 크기까지 이미지가 반복된다.
* repeat-x, repeat-y: x축,y축방향으로만 이미지를 반복시킨다.
* no-repeat : 이미지의 크기가 요소의 크기보다 작아도 이미지를 반복시키지 않는다

### background-position
* 배경 이미지의 위치를 지정한다.
* x좌표, y좌표로 지정할 땐 앞에오는 좌표가 x좌표 뒤에오는 좌표가 y좌표다
* 키워드를 사용해 지정할 땐 top,right,bottom,left를 순서에 상관없이 스페이싱을 이용해 지정한다. 하나의 키워드만 사용할땐 다른 하나의 값은 center로 지정된다.

### background-origin
* 배경의 원점을 border 시작점, border 내부, padding 내부 중 하나로 지정한다.
* padding-box : 기본값으로 배경의 원점이 border의 내부가 기준점이 된다.
* content-box : 배경의 원점이 padding의 내부가 기준점이 된다.
* border-box : 배경의 원점이 border의 시작점이 기준점이 된다. 

### background-size
* 배경 이미지의 크기를 지정한다. 
* auto : 기본값으로 이미지 원본의 크기로 지정한다.
* cover : 이미지가 찌그러지지 않는 한도 내에서 제일 크게 설정. 이미지의 가로세로비가 다르다면 더 긴부분을 잘라내어 빈 공간없이 요소에 맞게 꽉 채운다
* contain : 이미지가 잘리거나 찌그러지지 않는 한도 내에서 제일 크게 설정한다.
