# 그리드
* 그리드 레이아웃은 웹페이지를 위한 2차원 레이아웃 시스템이다.
* 수평선과 수직선으로 이루어져 있으며, 복잡한 레이아웃을 직관적으로 구축할 수있다.

##  1. grid-container에 사용하는 속성

### grid-template-rows, grid-template-columns
* grid-container에 몇개의 행과 열을 사용할지, 각 행과 열의 크기를 얼마로 할지 지정한다.
* fr단위를 사용하여 컨테이너 크기에 알맞게 행과 열에 크기를 나눌수 있다.
```
.container {
  height: 500px;
  border: 2px solid greenyellow;
  display: grid;
  grid-template-columns: 2fr 1fr 1fr;
  grid-template-rows: 2fr 2fr 1fr;
}
```
