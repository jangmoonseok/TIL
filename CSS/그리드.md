# 그리드
* 그리드 레이아웃은 웹페이지를 위한 2차원 레이아웃 시스템이다.
* 수평선과 수직선으로 이루어져 있으며, 복잡한 레이아웃을 직관적으로 구축할 수있다.

##  1. grid-container에 사용하는 속성

### grid-template-rows, grid-template-columns
* grid-container에 몇개의 행과 열을 사용할지, 각 행과 열의 크기를 얼마로 할지 지정한다.
* fr단위를 사용하여 컨테이너 크기에 알맞게 행과 열에 크기를 나눌수 있다.
```
.container {
  height: 500px;
  border: 2px solid greenyellow;
  display: grid;
  grid-template-columns: 2fr 1fr 1fr;
  grid-template-rows: 2fr 2fr 1fr;
}
```
![image](https://user-images.githubusercontent.com/64582209/133204866-aeb55dc2-d4ef-4165-b5c0-4a7b827272c2.png)
* 2:1:1의 비율의 크기를 가진 3열과 2:2:1의 비율의 크기를 가진 3행을 가진다.

### grid-template-areas
* grid-container를 여러개의 구역을 나누어 그 구역에 이름을 명시해 그 구역에 콘텐츠를 넣을때 사용
*  gird-area 속성을 사용해 각 아이템에게 들어갈 구역을 명시한다.
```
HTML
  <div class="container">
        <div class="item header">header</div>
        <div class="item main">main</div>
        <div class="item right_menu">right menu</div>
        <div class="item left_menu">left menu</div>
        <div class="item footer">footer</div>
    </div>
CSS
.container {
  height: 500px;
  border: 2px solid greenyellow;
  display: grid;
  grid-template-columns: repeat(5, 1fr);
  grid-template-rows: repeat(3, 1fr);
  grid-template-areas:
    "hd hd hd hd hd"
    "rm mn mn . lm"
    "ft ft ft ft ft";
}

.header {
  grid-area: hd;
}
.main {
  grid-area: mn;
}
.right_menu {
  grid-area: rm;
}
.left_menu {
  grid-area: lm;
}
.footer {
  grid-area: ft;
}
```

* 3행5열로 그리드의 전체적은 틀을 잡고 grid-template-areas로 세세한 구역을 

### row-gap, column-gap
* 각 행과 열사이 공백의 간격을 설정한다.

### grid-auto-rows, grid-auto-columns
* 아이템이 grid-template-rows, grid-template-columns로 설정한 템플릿을 벗어나 추가될때 그 아이템의 크기를  암시적으로 지정한다.

### grid-auto-flow
* 아이템이 자리를 찾아가는 방법을 지정한다.
* row : 기본 값으로 아이템이 행 방향으로 나열된다.
* columns : 아이템이 열 방향으로 나열된다.
* dense : template에 빈 공간이 있을때 코드 순서대로 나열하지않고 빈 공간을 꽉 채운다.

### grid
* grid-template-rows, grid-template-columns, grid-template-areas, grid-auto-rows, grid-auto-columns, grid-auto-flow 값을 한번에 설정가능한 단축 속성
* /로 나누어 앞에는 row에 관련된 값, 뒤는 column에 관련된 값을 작성한다.

### justify-content, align-content
* 작동 원리는 flex에서 배운 속성과 동일하다.
* 단 컨테이너의 크기가 콘텐츠의 크기보다 커서 빈 영역이 있을때 사용

### justify-items, align-items
* 행이나 열이 아닌 하나의 섹션안에서의 아이템의 정렬방법을 지정한다.
* 작동 원리는 flex에서 배운것과 동일하다.

## 2. grid-item에 사용하는 속성

### grid-row, grid-column
* start, end를 같이 사용하는 단축속성으로 아이템이 행과 열의 어디부터 어디까지 차지할지 지정하는 속성
```
.container {
  width: 500px;
  height: 500px;
  border: 2px solid greenyellow;
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: repeat(3, 1fr);
}

.item:first-child {
  grid-row: 1 / 4;
  background-color: aqua;
}

.item:nth-child(2) {
  background-color: bisque;
  grid-column: 2 / 4;
}
```
![image](https://user-images.githubusercontent.com/64582209/133219579-c8a55a98-1b27-49ae-b07f-fa77df42646d.png)
* 1번아이템은 1번부터 4번까지 행을 차지하고, 2번아이템은 2번부터 4번까지 열을 차지한다. 


### grid-area
* grid-template-areas로 지정한 구역의 이름에 아이템을 넣을때 사용
* grid-row-start / grid-column-start / grid-row-end / grid-column-end 순서로 작성해 단축속성으로도 사용

## 3. Grid 단위
* fr : 상대단위로 절대단위로 정의한 영역을 제외한 나머지 영역을 상대적인 비율에 따라 나누는 단위
* min-content : 콘텐츠에 들어간 단어중에 제일 긴 단어길이를 기준으로 길이를 지정
* max-content : 콘텐츠의 단어가 아닌 그 자체의 크기를 길이로 지정
* auto-fill : 반응형을 만들때 컨테이너 크기에 따라 가능한 범위 안에서 가장 많은 열 또는 행을 만들어준다.
* auto-fit : 기본적으로 auto-fill과 같이 작동하나 컨테이너의 크기가 콘텐츠의 크기보다 더 커질때 열 또는 행을 만들지 않고 콘텐츠의 크기를 컨테이너에 맞춰 꽉 채운다.
1. auto-fill
```
.container {
  border: 2px solid greenyellow;
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(100px, 1fr));
  grid-auto-rows: 100px;
}
```
![GIF 2021-09-14 오후 5-51-06](https://user-images.githubusercontent.com/64582209/133227041-163a1d50-9ba9-47ba-8589-2e2905441519.gif)
* 화면이 계속 늘어나면 기존 콘텐츠는 최소크기인 100px로 유지하고 그에 맞게 열을 계속 만든다.

2. auto-fit
```
.container {
  border: 2px solid greenyellow;
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
  grid-auto-rows: 100px;
}
```
