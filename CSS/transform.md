# transform
* 요소에 회전, 크기 조절, 기울이기, 이동 효과를 부여해 요소를 변경시킨다.
* 하나 이상의 변형 함수를 지정하면 요소에 변형을 오른쪽부터 왼쪽으로 순서대로 적용시킨다.

## scale()
* 2d평면에서 요소의 크기를 변경하고 인자로 받는 자료형은 number이다.
* width,height를 변경하는것과 달리 요소의 크기는 그대로 계산하고 눈에 보여지는 사이즈만 변경된다.
* 인자를 두개를 사용할 경우 앞에오는 인자가 x축 뒤에오는 인자가 y축

## rotate()
* 2d평면에서 요소의 원점을 기준으로 회전을 시킨다. 인자로 받는 자료형은 angle(deg, turn)이다.
* 한개의 인자만 받아 사용한다.

## translate()
* 2d평면에서 요소의 시작점을 기준으로 요소를 이동을 시킨다.인자로 받는 자료형은 length, percent이다
* 한개의 인자만 받아 사용될 경우 x축만 이동한다.
* 인자를 %로 입력받으면 viewport의 크기가 아니라  transform-box속성에 의해 정의된 참조 상자의 크기를 기준으로 움직인다.

## skew()
* 2d평면에서 요소를 기울인다. 인자로 받는 자료형은 angle(deg,turn)이다.
* 한개의 인자만 받아 사용될 경우 x축만 기울여진다.

## transform-origin
* transform을 시킬때 기준으로 할 원점을 지정한다. 기본값은 center로 요소의 중앙이다.
* 좌표계, 키워드(top,right,bottom,left,center)로 위치를 지정한다.
