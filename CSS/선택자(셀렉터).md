# 선택자의 종류

## 주요 선택자
```
예제 코드

 <h1 id="welcome-title">Welcome</h1>
    <h2>Movie List</h2>
    <ul>
        <li class="blue red">Toy Story</li>
        <li class="red">Zootopia</li>
        <li class="blue">Inside Out</li>
    </ul>
```
### 1. Type Selector(태그선택자, 유형선택자)
* 태그의 이름을 선택자로 CSS속성을 적용하는 방식
* 특정 요소를 위해서 Type Selector를 사용하지 않고 일관성있는 스타일링을 위해 공통적으로 사용할 코드를 작성할때 사용하는것을 권장
```
적용코드

h2{
    color:purple
}
```

## ID Selector
* html태그의 id속성의 값을 선택자로 CSS속성을 적용하는 방식
* 동일한 id속성 값을 가지면 안되고 유일해야한다.

```
적용코드

#welcome-title {
  color: red;
}
```

## Class Selector
* html태그의 class 속성의 값을 선택자로 CSS를 적용하는 방식
* ID Selector와 달리 여러개의 동일한 Class 값을 가져도 된다.
* 스페이싱을 사용해 두가지 이상의 Class 값을 가질수 있다.

```
적용코드
.blue {
  color: blue;
}

.red {
  color: red;
}
```
