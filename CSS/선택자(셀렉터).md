# 선택자의 종류

## 주요 선택자
```
예제 코드

 <h1 id="welcome-title">Welcome</h1>
    <h2>Movie List</h2>
    <ul>
        <li class="blue red">Toy Story</li>
        <li class="red">Zootopia</li>
        <li class="blue">Inside Out</li>
    </ul>
```
### 1. Type Selector(태그선택자, 유형선택자)
* 태그의 이름을 선택자로 CSS속성을 적용하는 방식
* 특정 요소를 위해서 Type Selector를 사용하지 않고 일관성있는 스타일링을 위해 공통적으로 사용할 코드를 작성할때 사용하는것을 권장
```
적용코드

h2{
    color:purple
}
```

## 2. ID Selector
* html태그의 id속성의 값을 선택자로 CSS속성을 적용하는 방식
* 동일한 id속성 값을 가지면 안되고 유일해야한다.

```
적용코드

#welcome-title {
  color: red;
}
```

## 3. Class Selector
* html태그의 class 속성의 값을 선택자로 CSS를 적용하는 방식
* ID Selector와 달리 여러개의 동일한 Class 값을 가져도 된다.
* 스페이싱을 사용해 두가지 이상의 Class 값을 가질수 있다.

```
적용코드
.blue {
  color: blue;
}

.red {
  color: red;
}
```

## 속성 선택자

```
예제코드
     <ul>
        <li><a href="http://example.com" target="_blink">Example Link(com/http)</a></li>
        <li><a href="http://example.org" target="_blink">Example Link(org/http)</a></li>
        <li><a href="https://example.com">Example Link(com/https)</a></li>
        <li><a href="https://example.org">Example Link(org/https)</a></li>
    </ul>
```

### 1.[attr]
* 태그의 속성이름을 선택자로 해당 속성 이름을 가진 태그에 CSS를 적용하는 방식
```
적용태그

a[target] {
  color: red;
}
```

### 2.[attr=value]
* 태그의 속성이름과 값을 선택자로 해당 속성-값을 가진 태그에 CSS를 적용하는 방식
```
적용태그

a[href="https://example.org"]
{
  color: purple;
}
```

### 3. [attr^=value]
* 태그의 속성-값을 가진 애들중 값이 value로 시작하는 애들 모두를 선택자로 CSS를 적용하는 방식
```
적용코드
a[href^="http"] {
  color: red;
}
a태그의 href속성을 가진 애들중 값이 http로 시작하는 애들 모두 선택

a[href^="https"] {
  color: green;
}
a태그의 href속성을 가진 애들중 값이 https로 시작하는 애들 모두 선택
```

### 4. [attr$=value]
* att^와 반대로 태그의 속성-값을 가진 애들중 값이 value로 끝나는 애들 모두를 선택자로 CSS를 적용하는 방식
```
적용코드
a[href$=".com"] {
  color: aqua;
}
a태그의 href라는 속성을 가진 애들중 값이 .com으로 끝나는 애들 모두 선택
```

### 5. [attr*=value]
* 태그의 속성-값을 가진 애들중 값에 value라는 문자열을 하나라도 가진 모든 애들을 선택자로 CSS를 적용하는 방식
```
적용코드
a[href*="example"] {
  color: brown;
}
a태그의 href라는 속성을 가진 애들중 값에 example이라는 문자열을 가진 모든 애들을 선택
```
