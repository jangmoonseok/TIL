# 폰트 관련 속성

* font-size : 글자의 크기를 정할때 사용
  * 숫자와 단위, 키워드로 값을 정한다.
  * 대부분의 브라우저에서 기본값은 16px이다.

*  font-style : 자주 사용하지않지만 italic, oblique 등 기울임꼴을 사용할 때 종종 사용된다.

* font-weight : 글자의 굵기를 정할때 사용
  * 보통은 100단위의 숫자를 사용해 굵기를 정한다.
  * 기본값은 normal로 숫자로는 400이다.
  * 아무 의미 없이 단순히 스타일링적으로 글씨는 굵게 보이게 하고 싶을때는 ```<strong>```을 사용하지 말고 bold를 사용한다.

* font-family : 글씨의 글꼴을 정할 때 사용
  * 여러개의 폰트를 나열해서 사용하며, 앞에 나열한 폰트가 없을 때 다음 폰트를 사용한다.

* line-height : 줄의 높이(줄고 줄사이 간격을 주고싶을때) 를 정할때 사용
  * 글꼴마다 기본 line-height가 다르기 때문에 가독성을 높이기 위해 값을 설정해주어 사용한다.

* font : font-size, font-family, font-style, font-variant, font-weight, line-height 의 단축속성이다.
 * font-size, font-family는 필수이며 그 외는 옵션이다.
 * 스페이싱으로 속성을 나열한다.
 * font-style, font-weight는 font-size보다 앞에 와야하고 line-height는 font-size 뒤에 /를 사용하여 입력한다.

*  text-align : block요소에서 여백이 남아있을때 text를 가운데,왼쪽,오른쪽 정렬을 하고싶을 때 사용한다.
 * inline요소안에 text를 가운데로 정렬하고 싶을땐 inline요소를 감싸는 부모block요소에 text-align: center라고 적용해준다.

* text-indent : block요소로 작성한 문단의 첫번째 줄에 들여쓰기를 하고싶을 때 사용한다.
* 
* text-decoration : CSS 단축 속성으로 글씨의 장식에 사용된다.
 * text-decoration-line(필수), text-decoration-color, text-decoration-style, text-decoration-thickness 속성을 단축시킨 속성
 * text-decoration-line : text에 line을 decoration 하고 싶을때 사용한다. 한가지값만 사용하는게 아니라 여러가지 속성을 동시에 사용가능하다.
 * text-decoration-style : line의 스타일을 지정한다. 
 * text-decoration-color : line의 색상을 지정한다.
 * text-decoration-thickness : line의 두께를 지정한다.
