# 단위와 값

## 숫자
* 일부 값은 단위를 추가하지 않고 숫자를 허용합니다. 
* 단위가 없는 숫자를 쓰는 속성의 예시로는 opacity, line-height(단위도 씀) 등이있다.

## 길이

* 절대 길이 단위 : 절대적인 단위로 viewport를 확대,축소해도 항상 같은 단위로 보여지기 때문에 접근성 문제를 유발할 수 있다.


  * px : 절대 길이 단위에서 제일 많이 사용하는 단위 1px = 1/96in

* 상대 길이 단위 : 상대적인 단위로 다른요소(viewport크기, 상위요소 글꼴크기)에 따라 비례하여 조정된다.

### 글꼴 상대 길이

  * em : 부모요소의 글자크기따라 상대적으로 비례하는 단위 (1.5em = 부모의 글자크기의 1.5배)

  * rem : 브라우저의 기본 글자크기따라 상대적으로 비례하는 단위(기본 글자크기 16px 일때 2rem = 32px) 

### 뷰포트 상대 길이

* 반응형 웹사이트를 만들때 주로 사용한다.
  * vw : viewport width 약자로 1vw는 viewport 너비의 1%이다.

  * vh : viewport height 약자로 1vh는 viewport 높이의 1%이다.
