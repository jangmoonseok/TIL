# object
* object는 다수의 원시 자료형을 포함하거나 복잡한 개체를 표현할 수 있는 자료형
* object는 Object()혹은 중괄호({})를 통해 생성
* object의 개체는 key:value형태로 표현하며, 접근은 object.key 형태로 표현
![image](https://user-images.githubusercontent.com/64582209/134798536-0a521af0-3fe2-4985-9c3c-8b99de26ccb7.png)

### 예제코드1
```
let user = {
    name:"john", //key: name, value:"john"
    age:27//key:age, value:27
}

console.log(typeof user)//object
console.log(typeof user.name)//string
console.log(typeof user.age)//number

console.log(user.name)//john
console.log(user.age)//27
user.age = 30
console.log(user.age)//30
```
### 예제코드2
* 객체에 개체 추가는 object.key = value, 삭제는 delete 명령어를 통해 수행
```
let user = {
    name:"john", //key: name, value:"john"
    age:27//key:age, value:27
}

console.log(user)//{ name: 'john', age: 27 }
user.weight = 72
console.log(user)//{ name: 'john', age: 27, weight: 72 }
delete user.age
console.log(user)//{ name: 'john', weight: 72 }
```

## 얕은복사

```

// 1. use for 
let admin = {}

for (let key in user){
    admin[key]=user[key]
}
 

// 2. use Object.assign 
let admin = Object.assign({},user)

// 3. use Spread Operator
let admin = {...user}


admin.name = 'park'
console.log(admin.name)//park
console.log(user.name)//john
user.age = 30
console.log(user.age)//30
console.log(admin.age)//27
```
## 얕은 복사의 문제점 : 객체 내 또 다른 객체가 있다면 복사되지 않음

```
let user = {
    name:"john", //key: name, value:"john"
    age:27,//key:age, value:27
    sizes:{
        height:180,
        weight:70
    }
}

let admin = {...user}

admin.sizes.height++
console.log(admin.sizes.height)//181
console.log(user.sizes.height)//admin.sizes.height만 변경했지만 user.sizes.height도 바뀜
```

## 깊은 복사
* 1. JSON객체를 이용한 깊은 복사, stringify는 객체를 문자열로 변한하는데 이때 원본 객체와의 참조가 끊김

```
let admin = JSON.parse(JSON.stringify(user))

admin.sizes.height++
console.log(admin.sizes.height)//181
console.log(user.sizes.height)//180
```

# 형 변환
* 자바스크립트는 느슨한 타입 언어 또는 동적 타입 언어로 변수의 자료형을 명시적으로 선언할 필요가 없는 언어
* 연산자로 인한 계산이나 변수에 전달되는 값은 자동으로 암묵적 형 변환 수행
* 강제적 형 변환을 위해서는 자료형 함수를 이용해 명시적 형 변환 수행

### String 형변환
```
console.log(String(123)) //123
console.log(String(1 / 0)) // Infinity
console.log(String(NaN)) // NaN
console.log(String(true)) // true
console.log(String(false)) // false
console.log(String(null)) // null
console.log(String(undefined)) // undefined
```
* type은 모두 String으로 바뀌고 출력값은 변함이 없다.


### Number 형변환
* Number는 정수와 실수를 모두 포함하는 자료 형 변환이므로, 정수 혹은 실수의 명시적 변환은 parse함수 사용
* 정수 변환:parseInt, 실수변환: parseFloat

```
console.log(Number("")) // 0
console.log(Number("123")) // 123
console.log(Number("123hello")) // NaN
console.log(Number("hello")) // NaN
console.log(Number(true))// 1
console.log(Number(false))// 0
console.log(Number(null))// 0
console.log(Number(undefined))// NaN
```
* 빈 문자열은 0으로 변환, true,false는 각각 1,0으로 변환 null은 빈 문자열과 같다고 판단

### Boolean 형변환
```
console.log(Boolean(""))// false
console.log(Boolean("123"))// true
console.log(Boolean("hello"))// true
console.log(Boolean("0"))// true
console.log(Boolean(NaN))// false
console.log(Boolean(0))// false
console.log(Boolean(123))// true
console.log(Boolean(null))// false
console.log(Boolean(undefined))// false
```
* 빈 문자열은 false로 변환 number는 0 을 제외한 실수는 true
