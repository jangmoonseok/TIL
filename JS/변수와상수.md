## 변수란?
* 변경 가능한 값을 저장하기 위한 기억공간
* 사용하기 전 반드시 선언해야함
* 변수명은 식별자를 이용해 정해야함
* 중복 선언 불가능
* 키워드:let

## 상수란?
* 변경 불가능한 값을 저장하기 위한 기억공간
* 사용하기 전 반드시 선언해야함
* 상수명은 식별자를 이용해 정해야함
* 상수명은 대문자로 표기하는것을 권장
* 중복 선언 불가능
* 선언과 동시에 값이 초기화
* 키워드:const

![image](https://user-images.githubusercontent.com/64582209/134796951-48bb1660-b594-4070-94ce-c7e2d576b88e.png)

### 식별자
* 스크립트에서 변수나 함수에 이름을 붙일 때 사용
* 대소문자를 구별하며 유니코드 문자셋을 이용
* 자바스크립트 내 식별자 규칙
  1. 키워드 사용 불가
  2. 숫자로 시작 불가
  3. 특수문자는 _와$만 허용
  4. 공백 문자 포함 불가

### 변수와상수 예제코드
```
// 선언 후 할당
let hi;
hi = "hello"
// 선언과 동시에 초기화
let halo = "hello!"

// 한 줄에 여러 변수 선언과 초기화
let name = "john", age = 12, msg = 'hello!!'

// 상수는 대문자로 표기
const TESTCASE = 5

console.log(hi)
console.log(halo)
console.log(name)
console.log(TESTCASE)
```
