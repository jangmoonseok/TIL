# 함수
* 함수는 다수의 명령문을 코드 블록으로 감싸고, 하나의 실행 단위로 만든 코드의 집합
* 유사한 동작을 하는 코드를 하나로 묶어, 범용성을 확대시킨 블록 코드
* 함수는 정의 부분과 호출 부분으로 구성
* 함수는 가급적 한가지 일만 하며, 매개 변수는 최대 3개 이내로 작성을 권장 

## 함수 정의
```
//1. 함수 선언식
function add(x,y){
    return x + y;
}

// 2. 함수 표현식
const add = function(x,y){
    return x + y;
}

// 3. 화살표 함수
const add = (x,y) => x + y;
```

## 함수 호출
* 자바스크립트 함수는 매개변수와 인수의 개수가 일치하는지 확인하지 않는다.
* ES6에서 도입된 기본값을 통해 매개변수에 undefined가 들어올 경우 값을 초기화 지정 가능
```
// 1. 기본 선언식
function print_add(x,y){
    console.log(x + y);
}

print_add(); // NaN
print_add(10) // NaN
print_add(10,20) // 30
print_add(10,20,30) //30

// 2. 매개변수 값 초기화 지정
function print_add(x,y = 10){
    console.log(x + y);
}

print_add(); // NaN
print_add(10) // 20
print_add(10,20) // 30
print_add(10,20,30) //30
```
* y = 10값을 초기화해 매개변수에 y값이 들어오지 않았을때에만 y = 10으로 지정하고, y값이 들어오면 들어온 값으로 지정된다.

## 재귀 함수
* 함수 스스로 자신을 참조해 호출하면서 동일한 코드가 계속적을 수행되는 함수 호출 방법
* 재귀 함수는 특정 조건이 됐을 때 자신을 그만 호출되도록 제한하는 exit code가 필요

```
function recursive(num){
    if(num === 0) return 0;
    return num + recursive(num - 1);
}
// 3 + (2 + (1 + 0))
console.log(recursive(3)) // 6

function factorial(x){
    if(x === 0) return 1;
    return x * factorial(x - 1);
}
// 4 * (3 * (2 * (1 * 1)))
console.log(factorial(4)) //24
```

## 콜백 함수
* 콜백 함수란 다른 함수의 매개변수로 전달되어 수행되어지는 함수
* 고차 함수란 매개변수를 통해 함수를 받아 호출하는 함수

![image](https://user-images.githubusercontent.com/64582209/134880196-081794a2-db42-4f8e-8fe9-eb49e58d4a55.png)
```
function add(x,y){
    return x + y;
}

function sub(x,y){
    return x - y;
}

function mul(x,y){
    return x * y;
}

function div(x,y){
    return x / y;
}

function calc(callback, x, y){
    return callback(x,y);
}

console.log(calc(add , 7, 3)) // add(7,3) => 7 + 3 = 10
console.log(calc(sub, 7, 3)) // sub(7,3) => 7 - 3 = 4
console.log(calc(mul, 7, 3)) // mul(7,3) => 7 * 3 = 21
console.log(calc(div, 7, 3)) // div(7,3) => 7 / 3 = 2.333333
```
* 사칙연산을 해주는 함수(콜백함수)를 calc함수(고차함수)의 매개변수로 받는다.

## call by *

### call by value
* 값에 의한 복사로 함수 내에서 매개 변수 값을 변경 시켜도 영향이 미치지 않음
* 원시 타입을 매개 변수로 넘겼을 때 발생

### call by reference
* 주소에 대한 복사로 함수 내에서 매개 변수 내 값을 변경시키면 원본 데이터에도 영향을 받음
* 객체 타입을 매개 변수로 넘겼을 때 발생

```
// call by value
let a = 1;
let add_1 = function (b) { b = b + 1;};
add_1(a);
console.log(a); // 1
/* add함수의 매개변수로 a를 넣어 b=a=1이 된다.
이때 b는 a의 값을 복사하고, add함수를 통해 b의 값은 2로 바뀌지만
원본 a의 값은 그대로 1이다.
*/

// call by reference
let x = {v:1};
let add_2 = function(y){y.v = y.v + 1;};
add_2(x);
console.log(x.v) // 2
/* x라는 변수에는 메모리 주소가 있고 그 메모리 주소안에 {v:1}이라는
객체가 들어있다. add함수의  매개변수로 x를 넣으면 y=x가 되는데, 이때
메모리 주소에 대한 복사가 일어난다. add함수로 인해 y.v가 2가되고, 
x는 y와 같은 주소를 가리키고 있기 때문에 x.v도 2가된다. */
```
