# object
* object는 다수의 원시 자료형을 포함하거나 복잡한 개체를 표현할 수 있는 자료형
* object는 Object()혹은 중괄호({})를 통해 생성
* object의 개체는 key:value형태로 표현하며, 접근은 object.key 형태로 표현
![image](https://user-images.githubusercontent.com/64582209/134798536-0a521af0-3fe2-4985-9c3c-8b99de26ccb7.png)

### 예제코드1
```
let user = {
    name:"john", //key: name, value:"john"
    age:27//key:age, value:27
}

console.log(typeof user)//object
console.log(typeof user.name)//string
console.log(typeof user.age)//number

console.log(user.name)//john
console.log(user.age)//27
user.age = 30
console.log(user.age)//30
```
