# object
* object는 다수의 원시 자료형을 포함하거나 복잡한 개체를 표현할 수 있는 자료형
* object는 Object()혹은 중괄호({})를 통해 생성
* object의 개체는 key:value형태로 표현하며, 접근은 object.key 형태로 표현
![image](https://user-images.githubusercontent.com/64582209/134798536-0a521af0-3fe2-4985-9c3c-8b99de26ccb7.png)

### 예제코드1
```
let user = {
    name:"john", //key: name, value:"john"
    age:27//key:age, value:27
}

console.log(typeof user)//object
console.log(typeof user.name)//string
console.log(typeof user.age)//number

console.log(user.name)//john
console.log(user.age)//27
user.age = 30
console.log(user.age)//30
```
### 예제코드2
* 객체에 개체 추가는 object.key = value, 삭제는 delete 명령어를 통해 수행
```
let user = {
    name:"john", //key: name, value:"john"
    age:27//key:age, value:27
}

console.log(user)//{ name: 'john', age: 27 }
user.weight = 72
console.log(user)//{ name: 'john', age: 27, weight: 72 }
delete user.age
console.log(user)//{ name: 'john', weight: 72 }
```

## 얕은복사

```

// 1. use for 
let admin = {}

for (let key in user){
    admin[key]=user[key]
}
 

// 2. use Object.assign 
let admin = Object.assign({},user)

// 3. use Spread Operator
let admin = {...user}


admin.name = 'park'
console.log(admin.name)//park
console.log(user.name)//john
user.age = 30
console.log(user.age)//30
console.log(admin.age)//27
```
## 얕은 복사의 문제점 : 객체 내 또 다른 객체가 있다면 복사되지 않음

```
let user = {
    name:"john", //key: name, value:"john"
    age:27,//key:age, value:27
    sizes:{
        height:180,
        weight:70
    }
}

let admin = {...user}

admin.sizes.height++
console.log(admin.sizes.height)//181
console.log(user.sizes.height)//admin.sizes.height만 변경했지만 user.sizes.height도 바뀜
```
