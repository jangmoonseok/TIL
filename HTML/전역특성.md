# 전역특성

### id
* id 는 문서 전체에서 유일한 고유 식별자를 정의한다. 스크립트 및 스타일링 적용할때 사용된다.
* -, _ , . 를 제외한 특수문자는 사용불가하고 영소문자로 시작돼야한다.
* id 특성의 값은 불투명 문자열로, 웹 작성자가 id 특성을 통해 사람이 읽을 수 있는 정보를 나타내서는 안된다.

### class
* class 공백으로 구분한 요소 클래스의 목록으로, 마찬가지로 스타일링 및 스크립트를 적용할때 사용한다.
* 대소문자 구분없이 사용하고, 공백으로 복수개의 클래스명칭을 넣을수 있다.
* 웹 작성자는 class 명칭을 요소의 의미나 목적으로 명칭하는것이 좋다.

### style
* 요소에 적용할 CSS 스타일 선언을 담는다.
* 이 특성은 주로 빠른 테스트 스타일링을 위한 목적으로 사용되며 스타일링은 외부에 별도에 파일로 정의하는 것이 권장된다.

### title
* 요소와 관련된 추가 정보를 제공하는 텍스트를 툴팁으로 나타낸다.
* 주로 보조기술을 위해```<iframe>```요소에 이름 붙이기,```<label>```을 표시하지 못할때에 대비해 input요소에 title 속성을 추가해서 사용한다.
* title속성이 없는 자식요소는 title속성이 있는 부모로부터 상속받는다.

### data - *
* 요소의 데이터특성을 사용자 눈에 보이지 않게 의미론적으로 저장할 수 있고 스크립트로 HTML과 DOM사이에서 교환하도록 해준다.
* data - "속성이름" 으로 작성한다.

### draggable
* 요소의 드래그 가능 여부를 나타내는 열거형 특성
* 기본값은 "auto"로 브라우저 기본 동작을 따르고 "true"는 드래그가능, "false"는 드래그 불가능
* 단독으로 HTML에서만 쓰일경우 아무런 의미가 없고 보통 JavaScript와 함께 사용된다. 

### hidden
* hidden을 추가하면 시각적으로 요소가 보이지 않지만 웹 페이지에서 사라지는것은 아니다.
* hidden속성을 가진 요소를 CSS의 display 속성 값을 변경하여 동작을 재정의 할 수있다.
