List
----------------
```<ol>```
* 정렬된 목록으로 아이템 앞에 숫자가 붙는다.
* 순위,요리 레시피 등 에 사용한다.
#### 속성
* type : 숫자, 영어 대문자, 영어 소문자, 로마숫자 등으로 표현할 때 사용
* start: 목록이 시작되는 순서를 변경할 때 사용
* reversed: 목록의 순서를 반대로 나타낼때 사용

```<ul>```
* 비정렬 목록으로 아이템 앞에 숫자가없다.
* ul태그가 중첩될 수록 깊이에 따라 Bullet Point모양이 바뀐다.
* ol과 중첩되어 사용가능

```<dl>```
* 설명 목록으로 ```<dt>```로 표기한 용어, ```<dd>```로 표기한 설명 그룹의 목록을 감싸서 사용
* 주로 용어사전이나 메타데이터를 표시할 때 사용
* 여러 개의 용어와 하나의 설명, 하나의 용어와 여러 개의 정의 로도 사용가능
* ```<dt>,<dd>```의 형제로 ```<div>```를 사용하지 않는게 웹 표준에 적합하다


Table
--------------
* 레이아웃을 위해 사용 되기도 하나 최대한 지양한다.


```<tr>```
* table row 라는 뜻으로 하나의 행을 나타낸다.

```<th>```
* 행이나 열의 제목으로 사용한다. 굵은 글씨로 표시됨
#### 속성
* scope : 웹 접근성을 높이기 위해 헤더에 속한 행이나 열을 대표하는 정보라고 정의 하기 위해 사용 
```<th scope="col">
이 헤더는 자신이 속한 열을 대표한다.
```
```<td>```
* table data 라는 뜻으로 테이블의 내용을 나타낸다.
#### 속성
* colspan: 셀이 차지하는 영역을 변경할 때 사용
