메타데이터의 역할
----------------
* 데이터를 설명하는 데이터
* 검색 엔진에서 문서 정보를 용이하게 수집하고 분류하게 도움을 주기 위한 정보

```<title>```
-----------------

* 브라우저의 제목 표시줄이나 페이지 탭에 보이는 문서 제목을 정의
* 문자열만 작성가능
* <head>태그 안에 하나의 <title>태그만 작성가능
* 페이지 제목은 페이지를 간략하게 설명하는 내용을 적는것이 좋음
  
※명사로만 작성하게 되면 검색엔진에서 순서를 내리곤 한다.

```<meta>```
------------------
## 속성
*  name : name과 content 특성을 함께 사용하면 문서의 메타데이터를 이름-값 쌍으로 제공할 수 있다. 
```
HTML에서 정의하는 표준 메타데이터 이름

* application-name:웹 페이지에서 구동중인 애플리케이션의 이름
* author: 문서 저작자
* description:페이지에 대한 짧고 정확한 요약 특정 브라우저에서 즐겨찾기 페이지의 기본 설명값으로 description을 사용한다.
* generator: 소프트웨어를 통해 페이지를 만들었을때  페이지를 생성한 소프트웨어의 식별자
* keyword:페이지의 콘텐츠와 관련된, 쉼표로 구분한 키워드 목록 인스타그램의 해쉬태그와 같은 용도
* referrer: 문서에서 시작하는 요청의 HTTP Referer 헤더를 통제
