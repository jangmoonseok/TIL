# 임베디드 요소
* 직접 코드로 작성하는게 아니라 외부에서 소스를 가져오는 요소


```<img>```
* 문서에 이미지를 넣는 빈 요소
* src특성은 필수이며, 이미지의 경로를 지정
* alt 특성은 이미지의 텍스트 설명이며 필수는 아니지만, 사용하면 접근성 차원에서 매우 유용
* width,height 속성은 픽셀 단위로 정수만 입력가능, width만 입력할 경우 가로 세로 비율에 맞춰 height도 자동 지정된다.
* srcset 속성은 반응형 이미지를 넣을때 여러개의 이미지 경로를 지정 하기위해 사용된다.(★테스트 용도로 사용할 이미지 파일을 만드는 사이트 'https://placeholder.com/')
```
웹에서 사용되는 이미지 유형 !
* 레스터 이미지
1. JPEG : 정지 이미지를 용량을 최대한 압축해서 이미지를 보여준다.
2. PNG : 원본 이미지를 보다 정확하게 보여주고 투명도가 필요한 경우 JPEG보다 선호된다.
3. GIF : 여러장의 이미지로 이루어진 애니메이션 표현 가능(JPEG,PNG 보다는 화질이 떨어짐)
4. WEBP : 구글이 만든 이미지 포맷, 압축률이 훨씬 우수하나 지원하는 브라우저가 제한적이다.
* 벡터 이미지
svg : 벡터와 벡터로된 다양한 크기로 정확하게 그려진 이미지로 선명한 화질을 보여주지만 작은 아이콘,UI이미지,다이어그램 등 작은 이미지로 사용된다.
```
```<video>```
* 비디오 플레이백을 지원하는 미디어 플레이어를 문서에 삽입
* ```<img>```와 달리 자식을 가질수 있는 요소로 자식요소는 브라우저가 ```<video>```를 지원하지 않을때 보여진다 
* src 속성을 사용하지 않고 자식에 ```<source>```태그를 사용해 비디오 url을 넣을수 있다.
* controls 속성은 소리 조절, 일시 정지/재시작을 할 수 있는 컨트롤러를 제공한다.
* autoplay 속성은 비디오가 자동으로 재생이 시작됩니다.(단, 소리가 있는 비디오는 새로고침을 하면 자동재생이 되지 않는다.)
* loop 속성은 동영상 재생이 끝나면 자동으로 처음으로 돌아간다.
* poster 속성은 비디오에 썸네일을 넣을때 사용한다.
