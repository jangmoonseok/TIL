```<h1>-<h6>```
-----------------
* 구획 제목을 나타내는 요소(heading content)
* ```<h1>```은 가장 중요한 전체 페이지의 목적을 설명하는 제목이므로 웹 접근성을 위해 페이지 당 하나의 ```<h1>```만 사용하는것을 권장
* 단순히 글씨 크기를 위해 제목 태그를 사용하지 말 것. 대신 CSS의 font-size 속성을 사용할 것

```<p>```
--------------
* block요소이기 때문에 차지할 수 있는 최대 너비를 가지고 안에 block요소나 inline요소를 포함할 수 있다.
* 두개 이상의 ```<p>```태그를 사용했을땐 기본적으로 문단사이 간격은 한줄간격이다. 들여쓰기를 하고싶거나 간격크기를 조절하고 싶을떈 CSS를 통해 사용할 것
* 단 자신을 닫는 태그```</p>```이전에 다른 block요소가 들어온다면 ```</p>```로 닫지 않아도 자동으로 닫힌다.

```<br>```
-------------
* HTML은 줄바꿈이나 스페이싱을 무시하기 때문에 줄바꿈을 위해 <br>을 사용
* 문단 사이에 여백을 두기 위한 용도로 사용하지말고 목적성에 맞게 줄바꿈 용도로만 사용하는것을 권장 문단 사이 여백을 주고싶을땐 CSS사용

인용문 요소 ```<blockqoute>,<q>```
-----------------
* ```<blockquote>``` :
  * 텍스트가 긴 인용문을 쓸때 사용
  *  기본 스타일링으로 들여쓰기가 들어감
  * ```<p>```태그내에서는 사용하지말 것


* ```<q>```
  * 문단 중간에 짧은 인용문을 쓸때 사용
  * 기본 스타일링으로 ""가 들어감

* cite 속성 : 인용문의 출처 URL을 남기기 위함
  * 브라우저만 출처정보를 알고 눈에는 보이지 않음

```<pre>```
------------
* 미리 서식을 지정한 텍스트를 나타내며, HTML에 작성한 내용 그대로 표현함(스페이싱, 엔터처리 가능)
* 고정폭글꼴 사용, 단독으로 사용가능

```<figure>```
--------------
* 독립적인 콘텐츠를 표현하기 위한 태그
* ```<blockqoute>, <q>, <img>, <pre>, <p>``` 등등 Flow content를 주로 감싼다
* 주로 2개이상의 문단이나 이미지-설명 등 서로 연관된 콘텐츠임을 나타낼때 사용
* ```<figcaption>```을 사용하여 설명을 포함하는것을 권장

```<hr>```
--------------
* 수평선을 그리는 빈요소
* 문단과 문단을 구분, 주제의 분리 용도로 사용
* 보통 단독으로 사용하지 않고 CSS로 스타일링을 해서 사용한다.

```<abbr>,<address>,<cite>```
---------------------
* ```<abbr>``` : 어떤 문장의 약어를 표현하는 의미의 태그 title속성과 함께 사용

* ```<address>``` : 이 태그 안에있는 텍스트는 주소를 의미담고있다.

* ```<cite>``` :  cite 속성과 달리 인용구의 출처를 화면에 나타낼때 사용


```<b>,<strong>```
-----------------
* 공통적으로 텍스트를 '굵은 글씨'로 나타낸다

* 단순히 스타일링을 위해 사용하지말 것

* ```<b>``` :  특별한 중요성을 가지고 있진 않지만 특정부분을 굵게 표현하고싶을때 사용

* ```<strong>``` : ```<b>```와 보여지는 스타일링은 같지만 높은 중요도를 나타내고싶을때 사용


```<i>,<em>```
------------------
* 공통적으로 기울임꼴로 표시
* ```<i>``` : 텍스트에서 어떤 이유든 주위와 구분해야 하는 부분을 나타낼때 사용
  * 예로 기술 용어, 외국어 구절, 등장인물의 생각 등

* ```<em>``` : 텍스트의 강세를 나타낼때 사용
  * ```<strong>```은 아주 중요한 내용을 강조할때 사용 ```<em>```은 강조 하고싶은 부분을 강조할때 사용


```<del>,<ins>,<code>```
----------------------
* ```<del>``` : 문서에서 제거, 삭제된 부분의 범위를 나타내는 요소

* ```<ins>``` : 문서에서 추가된 텍스트의 범위를 나타내는 요소

* ```<code>``` : 인라인 요소로 짧은코드를 나타낼때 사용
  * 여러줄의 코드를 나타내려면 <pre>로 감싸서 사용할 것
  
  
```<a>```
--------------
* href 특성을 통해 다른 페이지나 같은 페이지의 어느 위치, 파일, 이메일 주소 그 외 다른 URL로 연결할 수 있는 하이퍼링크를 만들때 사용

* href
  * 절대경로: 현재의 위치와 관계없이 무조건 그 경로로 이동한다

  * 상대경로: 현재있는 위치 기준으로 이동 하려고 하는 경로를 나타냄

  *이메일/전화 : href 안에 이메일은 ```mailto:```뒤에 이메일주소를 작성하고
  전화는 ```tel:``` 뒤에 전화 번호를 작성하면 이메일과 전화를 걸수 있다.
  
* target
  * 링크한 URL을 표시할 위치, 즉 표시할 창을 표시
  * _self : URL을 현재 탭에 표시한다. 기본값
  * _blank: URL을 새로운 탭에 표시한다. 보통 새 탭이지만, 브라우저 설정을 통해 새 창으로 변경가능
  

Entity
-----------------
* <, >, & 등의 특수문자는 HTML 구문 자체의 일부여서 코드로 해석된다. 그래서 표시를 할 수가 없기 때문에 코드로 해석되지 않고 표시되게 하고싶을때 사용

* &lt; : <
* &gt; : >
* &quot; : "
* &apos; : '
* &amp; : &
* &nbsp; 스페이스바
