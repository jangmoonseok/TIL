빈 요소(Empty elements)
------------------
* 내용이 없는 요소를 빈 요소라 부른다
* 이 경우 닫는 태그를 추가로 명시할 필요 없음
* <br>, <hr>, <img>, <meta>, <input> 등등
* <br/>처럼 /를 쓰는것도 문법적으로 틀리지 않지만 일관적으로 작성하는게 좋다.

요소의 중첩(Nesting)
------------------
* 요소 안에 다른 요소가 들어가는 포함관계를 성립할 수 있다.
* 여러 요소가 중첩될 경우에는, 열린 순서 반대로 닫혀야 한다.
* 서로의 포함관계(부자관계)를 구분하기 위하여 들여쓰기를 사용한다.
```
<ul>
  <li>하나</li>
  <li>둘</li>  
  <li>셋</li>  
</ul>

ul태그가 부모요소 li태그가 자식요소
부모 자식을 구분하기위해 li태그에 들여쓰기 사용
```
